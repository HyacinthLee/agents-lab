# 命令行计算器 PRD

## 1. 产品概述
一款轻量级命令行计算器工具，支持基础四则运算并提供完善的错误处理机制，满足用户快速、准确的计算需求。

## 2. 功能列表

| 功能 | 描述 |
|------|------|
| 加法运算 | 支持两个或多个数字相加 |
| 减法运算 | 支持两个数字相减 |
| 乘法运算 | 支持两个或多个数字相乘 |
| 除法运算 | 支持除法计算，自动检测除零错误 |
| 错误处理 | 除零检查、非法输入提示、运算符校验 |

## 3. 验收标准

1. **功能验证**：输入 `calc 10 + 5` 输出 `15`；输入 `calc 10 / 0` 提示错误"除数不能为零"
2. **错误处理**：非法输入（如 `calc 10 & 5`）返回清晰的错误提示信息
3. **边界测试**：支持整数和浮点数运算，精度保留小数点后6位

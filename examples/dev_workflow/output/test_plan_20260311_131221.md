# 命令行计算器测试计划

## 1. 测试范围

| 模块 | 测试内容 |
|------|----------|
| 加法运算 | 两个数相加、多个数相加、整数与浮点数混合 |
| 减法运算 | 两个数相减、负数结果处理 |
| 乘法运算 | 两个数相乘、多个数相乘 |
| 除法运算 | 正常除法、除零错误处理 |
| 错误处理 | 非法运算符、非法输入格式、参数缺失 |
| 精度验证 | 浮点数运算精度（小数点后6位） |

## 2. 测试策略

| 类型 | 策略说明 |
|------|----------|
| 正常场景 | 验证标准四则运算功能，覆盖整数和浮点数 |
| 异常场景 | 验证除零错误、非法运算符、无效输入的处理 |
| 边界条件 | 验证大数运算、小数精度、负数运算、零值运算 |

## 3. 测试用例列表

| 编号 | 标题 | 前置条件 | 操作步骤 | 期望结果 |
|:----:|------|----------|----------|----------|
| TC01 | 整数加法运算 | 计算器已安装 | 输入 `calc 10 + 5` | 输出 `15` |
| TC02 | 浮点数加法运算 | 计算器已安装 | 输入 `calc 3.5 + 2.7` | 输出 `6.2` |
| TC03 | 多个数字相加 | 计算器已安装 | 输入 `calc 1 + 2 + 3 + 4` | 输出 `10` |
| TC04 | 整数减法运算 | 计算器已安装 | 输入 `calc 20 - 8` | 输出 `12` |
| TC05 | 负数结果减法 | 计算器已安装 | 输入 `calc 5 - 10` | 输出 `-5` |
| TC06 | 整数乘法运算 | 计算器已安装 | 输入 `calc 6 * 7` | 输出 `42` |
| TC07 | 浮点数乘法精度 | 计算器已安装 | 输入 `calc 0.1 * 0.2` | 输出 `0.020000`（保留6位小数） |
| TC08 | 正常除法运算 | 计算器已安装 | 输入 `calc 15 / 4` | 输出 `3.75` |
| TC09 | 除零错误处理 | 计算器已安装 | 输入 `calc 10 / 0` | 提示错误"除数不能为零" |
| TC10 | 非法运算符处理 | 计算器已安装 | 输入 `calc 10 & 5` | 返回错误提示"非法运算符" |
| TC11 | 参数缺失处理 | 计算器已安装 | 输入 `calc 10 +` | 返回错误提示"参数不完整" |
| TC12 | 非数字输入处理 | 计算器已安装 | 输入 `calc abc + 5` | 返回错误提示"非法输入" |
| TC13 | 大数运算测试 | 计算器已安装 | 输入 `calc 999999999 * 999999999` | 正确输出结果 `999999998000000001` |
| TC14 | 零值运算测试 | 计算器已安装 | 输入 `calc 0 + 0` 和 `calc 5 * 0` | 分别输出 `0` 和 `0` |
| TC15 | 混合运算类型 | 计算器已安装 | 输入 `calc 10.5 + 5.5 - 3.0` | 输出 `13.0` |

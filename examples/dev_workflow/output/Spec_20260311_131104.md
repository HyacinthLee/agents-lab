# 命令行计算器软件规格说明书

## 1. 系统架构设计

本工具采用**分层架构设计**：解析层负责命令行参数解析与验证，计算层执行核心运算逻辑，输出层处理结果格式化与错误信息展示。各层通过标准数据接口通信，确保模块间低耦合、高内聚。

## 2. 模块划分

| 模块 | 职责 |
|------|------|
| **CLI解析器** | 解析命令行参数，识别运算符与操作数，验证参数数量与格式 |
| **运算引擎** | 执行加减乘除运算，支持整数与浮点数计算，保证6位小数精度 |
| **错误处理器** | 检测除零、非法运算符、参数缺失等异常，返回标准化错误信息 |
| **格式化输出** | 统一处理计算结果与错误提示的输出格式 |

## 3. 接口定义

### 3.1 CLI解析器
```python
def parse_args(args: List[str]) -> Tuple[List[float], str]:
    """
    输入: 命令行参数列表 (如 ["10", "+", "5"])
    输出: (操作数列表, 运算符) 或抛出 ValueError
    """
```

### 3.2 运算引擎
```python
def calculate(operands: List[float], operator: str) -> float:
    """
    输入: 操作数列表, 运算符
    输出: 计算结果 (float)
    异常: ZeroDivisionError, ValueError
    """
```

### 3.3 错误处理器
```python
def handle_error(error: Exception) -> str:
    """
    输入: 异常对象
    输出: 用户友好的错误提示字符串
    """
```

### 3.4 主入口
```python
def main(argv: List[str]) -> int:
    """
    输入: 命令行参数
    输出: 退出码 (0=成功, 1=错误)
    """
```

## 4. 数据结构

### 4.1 核心类型定义
```python
from typing import List, Tuple, Union
from enum import Enum

# 支持的运算符枚举
class Operator(Enum):
    ADD = "+"
    SUB = "-"
    MUL = "*"
    DIV = "/"

# 操作数类型 (支持整数和浮点数)
Operand = Union[int, float]

# 解析结果类型
ParseResult = Tuple[List[Operand], Operator]
```

### 4.2 错误码定义
| 错误码 | 说明 |
|--------|------|
| `E_INVALID_OP` | 非法运算符 |
| `E_DIV_ZERO` | 除数为零 |
| `E_ARG_COUNT` | 参数数量错误 |
| `E_TYPE_ERROR` | 操作数类型非法 |

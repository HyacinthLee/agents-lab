# 命令行计算器测试计划

## 1. 测试范围

| 模块 | 测试内容 |
|------|----------|
| 加法运算 | 两个数相加、多个数相加 |
| 减法运算 | 两个数相减 |
| 乘法运算 | 两个数相乘、多个数相乘 |
| 除法运算 | 正常除法、除零保护 |
| 错误处理 | 非法输入、参数缺失、运算符错误 |

## 2. 测试策略

| 测试类型 | 说明 |
|----------|------|
| 正常场景 | 标准输入格式，验证计算结果正确性 |
| 异常场景 | 非法字符、除零、参数不足等错误处理 |
| 边界条件 | 大数、负数、零值、浮点数运算 |

## 3. 测试用例列表

| 编号 | 标题 | 前置条件 | 操作步骤 | 期望结果 |
|:----:|------|----------|----------|----------|
| TC01 | 加法运算-两个正整数 | 计算器已部署 | 输入 `calc 10 + 5` | 输出 `15` |
| TC02 | 加法运算-多个数字 | 计算器已部署 | 输入 `calc 1 + 2 + 3 + 4` | 输出 `10` |
| TC03 | 减法运算-正整数 | 计算器已部署 | 输入 `calc 20 - 8` | 输出 `12` |
| TC04 | 减法运算-负数结果 | 计算器已部署 | 输入 `calc 5 - 10` | 输出 `-5` |
| TC05 | 乘法运算-两个正整数 | 计算器已部署 | 输入 `calc 6 * 7` | 输出 `42` |
| TC06 | 乘法运算-多个数字 | 计算器已部署 | 输入 `calc 2 * 3 * 4` | 输出 `24` |
| TC07 | 除法运算-正常整除 | 计算器已部署 | 输入 `calc 100 / 4` | 输出 `25` |
| TC08 | 除法运算-除零保护 | 计算器已部署 | 输入 `calc 10 / 0` | 提示错误信息，程序不崩溃 |
| TC09 | 浮点数运算 | 计算器已部署 | 输入 `calc 10.5 + 2.5` | 输出 `13.0` |
| TC10 | 非法输入-字母 | 计算器已部署 | 输入 `calc a + b` | 返回中文错误提示：非法输入 |
| TC11 | 非法输入-特殊字符 | 计算器已部署 | 输入 `calc 5 @ 3` | 返回中文错误提示：非法运算符 |
| TC12 | 参数缺失-缺少操作数 | 计算器已部署 | 输入 `calc 5 +` | 返回中文错误提示：参数不足 |
| TC13 | 参数缺失-缺少运算符 | 计算器已部署 | 输入 `calc 5 3` | 返回中文错误提示：格式错误 |
| TC14 | 边界值-大数运算 | 计算器已部署 | 输入 `calc 999999999 + 1` | 输出 `1000000000` |
| TC15 | 混合运算-负数参与 | 计算器已部署 | 输入 `calc -5 + 3` | 输出 `-2` |

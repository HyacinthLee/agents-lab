# 命令行计算器 PRD

## 1. 产品概述

一款简洁高效的命令行计算器工具，支持基础四则运算，具备完善的错误处理机制，为用户提供可靠的计算体验。

## 2. 功能列表

| 功能 | 描述 |
|------|------|
| 加法运算 | 支持两个或多个数字相加 |
| 减法运算 | 支持两个数字相减 |
| 乘法运算 | 支持两个或多个数字相乘 |
| 除法运算 | 支持两个数字相除，含除零保护 |
| 错误处理 | 检测除零、非法输入等异常情况 |

## 3. 验收标准

1. **功能正确性**：输入 `calc 10 + 5` 输出 `15`，输入 `calc 10 / 0` 提示错误且不崩溃
2. **错误处理**：非法输入（如 `calc a + b`）返回清晰的中文错误提示
3. **命令格式**：支持 `calc <数字> <运算符> <数字>` 的标准命令格式

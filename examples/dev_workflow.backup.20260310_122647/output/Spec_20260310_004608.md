# 软件规格说明书

## 1. 系统架构设计

### 模块划分

| 模块名称 | 职责描述 |
|---------|---------|
| `cli` | 命令行参数解析、主入口控制、交互模式管理 |
| `parser` | 表达式词法分析、语法分析、生成AST |
| `evaluator` | AST求值、运算符执行、函数调用 |
| `functions` | 数学函数库（三角函数、对数、幂运算等） |
| `converter` | 进制转换、单位换算 |
| `history` | 历史记录读写、持久化存储 |
| `variable` | 变量存储、作用域管理 |
| `error` | 错误类型定义、错误处理 |

### 模块间接口定义

| 接口 | 输入 | 输出 | 说明 |
|-----|------|------|------|
| `cli.parse_args()` | `sys.argv` | `Args` 对象 | 解析命令行参数 |
| `cli.run_interactive()` | 无 | 无 | 启动REPL交互模式 |
| `parser.parse(expression)` | 字符串表达式 | `ASTNode` | 将表达式解析为AST |
| `evaluator.evaluate(ast)` | `ASTNode` | `Number` | 计算AST结果 |
| `functions.call(name, args)` | 函数名, 参数列表 | `Number` | 执行数学函数 |
| `converter.convert(value, from, to)` | 数值, 源进制, 目标进制 | 字符串 | 进制转换 |
| `history.add(record)` | `HistoryRecord` | 无 | 添加历史记录 |
| `history.get_all()` | 无 | `List[HistoryRecord]` | 获取所有历史 |
| `variable.set(name, value)` | 变量名, 数值 | 无 | 设置变量 |
| `variable.get(name)` | 变量名 | `Number` | 获取变量值 |

## 2. 数据结构定义

### Token（词法单元）

```python
class Token:
    type: TokenType      # 词法单元类型
    value: str           # 词法单元值
    position: int        # 在表达式中的位置
```

### TokenType（词法单元类型枚举）

```python
enum TokenType:
    NUMBER          # 数字（整数或浮点数）
    PLUS            # + 加号
    MINUS           # - 减号
    MULTIPLY        # * 乘号
    DIVIDE          # / 除号
    POWER           # ^ 幂运算
    LPAREN          # ( 左括号
    RPAREN          # ) 右括号
    COMMA           # , 逗号
    IDENTIFIER      # 标识符（函数名、变量名）
    TO              # to 关键字（进制转换）
    BIN_PREFIX      # 0b 二进制前缀
    OCT_PREFIX      # 0o 八进制前缀
    HEX_PREFIX      # 0x 十六进制前缀
    EOF             # 结束符
```

### ASTNode（抽象语法树节点）

```python
class ASTNode:
    pass

class NumberNode(ASTNode):
    value: float | int      # 数值
    is_integer: bool        # 是否为整数

class BinaryOpNode(ASTNode):
    left: ASTNode           # 左操作数
    operator: TokenType     # 运算符
    right: ASTNode          # 右操作数

class UnaryOpNode(ASTNode):
    operator: TokenType     # 运算符（正负号）
    operand: ASTNode        # 操作数

class CallNode(ASTNode):
    func_name: str          # 函数名
    arguments: List[ASTNode] # 参数列表

class ConvertNode(ASTNode):
    value: ASTNode          # 待转换的值
    target_base: str        # 目标进制（bin/oct/dec/hex）
```

### Number（数值类型）

```python
class Number:
    value: float | int      # 实际数值
    display_format: str     # 显示格式（dec/bin/oct/hex/float）
    precision: int          # 小数精度（默认6位）
```

### HistoryRecord（历史记录）

```python
class HistoryRecord:
    id: int                 # 记录ID
    expression: str         # 原始表达式
    result: str             # 计算结果
    timestamp: datetime     # 时间戳
    success: bool           # 是否成功
    error_message: str      # 错误信息（失败时）
```

### Variable（变量）

```python
class Variable:
    name: str               # 变量名
    value: Number           # 变量值
    created_at: datetime    # 创建时间
```

### CalcError（错误类型基类）

```python
class CalcError(Exception):
```

## 3. 核心算法说明

### 3.1 表达式解析算法（递归下降解析）

采用递归下降解析器，支持运算符优先级：

```
优先级（从高到低）：
1. 括号、函数调用、进制转换
2. 正负号（一元运算符）
3. 幂运算 ^
4. 乘除 * /
5. 加减 + -
```

**伪代码：**

```
function parse():
    return parse_add_sub()

function parse_add_sub():
    left = parse_mul_div()
    while current_token in [PLUS, MINUS]:
        op = current_token
        advance()
        right = parse_mul_div()
        left = BinaryOpNode(left, op, right)
    return left

function parse_mul_div():
    left = parse_power()
    while current_token in [MULTIPLY, DIVIDE]:
        op = current_token
        advance()
        right = parse_power()
        left = BinaryOpNode(left, op, right)
    return left

function parse_power():
    left = parse_unary()
    if current_token == POWER:
        advance()
        right = parse_power()  # 右结合
        left = BinaryOpNode(left, POWER, right)
    return left

function parse_unary():
    if current_token in [PLUS, MINUS]:
        op = current_token
        advance()
        operand = parse_unary()
        return UnaryOpNode(op, operand)
    return parse_primary()

function parse_primary():
    if current_token == NUMBER:
        node = NumberNode(current_token.value)
        advance()
        return node
    if current_token == IDENTIFIER:
        return parse_identifier()
    if current_token == LPAREN:
        advance()
        node = parse_add_sub()
        expect(RPAREN)
        return node
    error("Unexpected token")
```

### 3.2 AST求值算法

```
function evaluate(node):
    match node.type:
        case NumberNode:
            return node.value
        
        case BinaryOpNode:
            left = evaluate(node.left)
            right = evaluate(node.right)
            return apply_binary_op(node.operator, left, right)
        
        case UnaryOpNode:
            operand = evaluate(node.operand)
            return apply_unary_op(node.operator, operand)
        
        case CallNode:
            args = [evaluate(arg) for arg in node.arguments]
            return call_function(node.func_name, args)
        
        case ConvertNode:
            value = evaluate(node.value)
            return convert_base(value, node.target_base)

function apply_binary_op(op, left, right):
    match op:
        case PLUS:  return left + right
        case MINUS: return left - right
        case MULTIPLY: return left * right
        case DIVIDE: 
            if right == 0: raise DivisionByZeroError
            return left / right
        case POWER: return pow(left, right)
```

### 3.3 进制转换算法

```
function convert_base(value, target_base):
    match target_base:
        case "bin":
            if is_integer(value):
                return "0b" + binary(value)
            else:
                raise Error("Cannot convert float to binary")
        case "oct":
            return "0o" + octal(int(value))
        case "hex":
            return "0x" + hexadecimal(int(value))
        case "dec":
            return str(float(value))
```

## 4. 错误处理策略

### 可能出现的错误类型

| 错误类型 | 错误代码 | 描述 | 示例 |
|---------|---------|------|------|
| `DivisionByZeroError` | E001 | 除数为零 | `1 / 0` |
| `SyntaxError` | E002 | 表达式语法错误 | `1 + * 2` |
| `InvalidTokenError` | E003 | 无效的词法单元 | `1 + $` |
| `UndefinedVariableError` | E004 | 未定义的变量 | `x + 1` |
| `UndefinedFunctionError` | E005 | 未定义的函数 | `foo(1)` |
| `TypeError` | E006 | 类型错误 | 浮点转二进制 |
| `ArgumentCountError` | E007 | 参数数量不匹配 | `sin(1, 2)` |
| `DomainError` | E008 | 数学定义域错误 | `sqrt(-1)` |
| `OverflowError` | E009 | 数值溢出 | 结果过大 |
| `InvalidExpressionError` | E010 | 无效表达式 | `abc` |
| `IOError` | E011 | 文件IO错误 | 历史文件读写失败 |

### 处理方式

1. **解析阶段错误**：捕获并报告具体位置和错误类型
2. **运行时错误**：在求值过程中捕获，返回友好的错误信息
3. **交互模式错误**：显示错误后继续等待输入
4. **批处理模式错误**：记录错误并继续处理下一行
5. **所有错误**：记录到日志（调试模式），但不暴露内部实现细节
